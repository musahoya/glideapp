## 📁 파일에 들어있는 시트 설명 (아주 중요)

### ① `Accounts`

* 구글 아이디 관리
* **아이디 추가 = 행 추가**
* `account_id`는 절대 수정 ❌

### ② `Services`

* 가입한 사이트 + 일일 무료 한도
* 사이트는 **여기만 등록**하면 전체 자동 반영

### ③ `Account_Service`

* 어떤 구글 아이디가 어떤 사이트에 가입했는지 체크
* TRUE / FALSE 로 관리

### ④ `Daily_Usage` ⭐ 핵심 입력 시트

* **매일 여기만 입력**
* 입력 항목:

  * 날짜
  * account_id
  * service_id
  * 사용량
* 👉 총할당량 / 남은량은 자동 계산

### ⑤ `Dashboard`

* 오늘 날짜 기준
* 아이디별 남은 무료량 요약용 (확장 예정)

# 🚀 1단계: 엑셀 → Google Sheets 업로드

⚠️ Glide는 **Google Sheets만 직접 연결 가능**합니다.

### 방법

1. **Google Drive** 접속
2. `새로 만들기 → 파일 업로드`
3. 👉 `구글아이디_무료할당량_관리템플릿.xlsx` 업로드
4. 업로드된 파일 **우클릭 → Google Sheets로 열기**
5. 자동 변환 완료

👉 이제 Glide에서 선택 가능

---

# 🚀 2단계: Glide 앱 생성

1. 👉 [https://www.glideapps.com](https://www.glideapps.com) 접속
2. Google 계정으로 로그인
3. **New App** 클릭
4. **From Google Sheets** 선택
5. 방금 변환한 시트 선택
6. App type: **Classic App** (처음엔 이게 가장 안정적)

# 🧠 Glide Computed Column 표준 세트 (중요)

## ① 남은비율

```
남은량 ÷ 총할당량
```

## ② 상태 텍스트

```
IF 남은비율 ≤ 0.2 → "위험"
IF ≤ 0.5 → "주의"
ELSE → "안정"
```

## ③ 상태 색상

* 위험 → 빨강
* 주의 → 노랑
* 안정 → 초록

👉 모든 화면에서 공통 사용
