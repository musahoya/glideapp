---

## Step 1: Clappia 가입 (1분)

1. https://www.clappia.com 열기
2. **Sign Up Free** 클릭
3. **Continue with Google** 선택
4. Google 계정 선택
5. Workplace 이름: "내관리앱" 입력
6. **Create Workplace** 클릭

✅ 완료!

---

## Step 2: Excel 파일 업로드 (1분)

### 방법: AI Assistant 사용

1. 왼쪽 메뉴에서 **AI Assistant** (🤖 아이콘) 클릭

2. 채팅창 하단 **+ 버튼** 클릭

3. **파일 선택**:
   - `구글아이디_무료할당량_관리_Clappia템플릿.xlsx` 선택
   - **Upload** 클릭

4. 채팅에 입력:
   ```
   Create apps from this Excel file
   ```

5. **전송** → 자동으로 앱 생성 시작!

⏱️ 30초 정도 기다리면 완료됩니다.

---

## Step 3: 앱 확인 (1분)

왼쪽 메뉴 **Apps**에서 생성된 앱 확인:

```
✅ Accounts
✅ Services
✅ Account_Service
✅ Daily_Usage          ← 메인 앱
✅ 사용가이드
```

**Daily_Usage** 앱을 클릭하세요!

---

## Step 4: 첫 사용량 입력 (2분)

### 4-1. 샘플 데이터 확인

**Daily_Usage** 앱 화면에서:
- 이미 **샘플 데이터** 3일치가 보임
- 표 형태로 날짜, 아이디, 사이트, 사용량 등 표시

### 4-2. 새 데이터 입력

1. 오른쪽 상단 **+ Add New** 클릭

2. 폼이 열리면 입력:
   ```
   날짜: [오늘 날짜 선택]
   account_id: [드롭다운에서 선택]
   service_id: [드롭다운에서 선택]
   총할당량: [자동 입력됨 - 확인만]
   사용량: 10
   남은량: [자동 계산됨]
   상태: [자동 판정됨]
   ```

3. **Submit** 클릭

🎉 **첫 데이터 입력 완료!**

## 🆘 문제 발생 시

### Q1: 앱이 생성되지 않아요
**A**: AI Assistant 대신 수동 업로드 시도
1. 왼쪽 **More** → **Create App from Excel**
2. 파일 업로드

### Q2: 드롭다운이 비어있어요
**A**: Accounts/Services 앱 먼저 확인
1. Accounts 앱에 아이디 있는지 확인
2. Services 앱에 사이트 있는지 확인

### Q3: 남은량이 자동 계산 안 돼요
**A**: Calculated Field 설정 필요
→ CLAPPIA_GUIDE.md의 "앱 설정" 섹션 참고

## 🔥 핵심 팁 3가지

### 1️⃣ 드롭다운이 편함의 핵심
- 아이디/사이트를 **직접 입력 절대 금지**
- 항상 **드롭다운 선택**만 사용
- → 오타 방지, 속도 10배 빠름

### 2️⃣ 매일 아침 30초 루틴
```
1. 앱 열기
2. + Add New
3. 드롭다운 선택 2번
4. 사용량 입력
5. Submit
```

### 3️⃣ 구글 아이디 늘어도 OK
- Accounts 앱에 **행 추가만** 하면 끝
- 앱 구조 수정 불필요
- 10개 → 100개도 가능
