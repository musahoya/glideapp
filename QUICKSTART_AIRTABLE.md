# ⚡ Airtable 3분 시작 가이드

> **가장 빠르게 시작하는 방법**

---

## ✅ 체크리스트

- [ ] Airtable 가입 (1분)
- [ ] Base 생성 (1분)
- [ ] 첫 데이터 입력 (1분)

**총 3분!**

---

## 1️⃣ 가입 (1분)

### 웹사이트 접속
```
https://airtable.com
```

### 가입
1. **"Sign up for free"** 클릭
2. **"Continue with Google"** 선택
3. Google 계정 선택
4. 완료!

---

## 2️⃣ Base 만들기 (1분)

### Base 생성
1. 대시보드에서 **"Start from scratch"** 클릭
2. Base 이름: `구글 아이디 관리`
3. 기본 테이블 이름 변경: `Accounts`

### 필드 설정 (클릭 몇 번!)

**Accounts 테이블:**

| 필드명 | 타입 | 설정 |
|--------|------|------|
| account_id | Single line text | - |
| email | Email | - |
| memo | Single line text | - |
| active | Checkbox | Default: ✅ |

**설정 방법:**
- 컬럼 헤더 클릭 → Customize field type → 타입 선택

---

## 3️⃣ 데이터 입력 (1분)

### 첫 행 입력:

```
account_id: A01
email: your-email@gmail.com
memo: 주력 계정
active: ✅
```

**방법:** 그냥 셀 클릭하고 타이핑! (엑셀처럼)

---

## 🎉 완료!

이제 기본 Airtable Base가 생성되었습니다!

---

## 📱 다음 단계 (선택)

### 즉시 해볼 것:

1. **모바일 앱 설치**
   - App Store / Google Play → "Airtable"
   - 로그인하면 자동 동기화!

2. **두 번째 테이블 추가: Services**
   - 왼쪽 하단 **"+ Add table"** 클릭
   - 이름: `Services`
   - 필드 추가:
     - service_id (Single line text)
     - site_name (Single line text)
     - daily_limit (Number)

3. **Form 만들기**
   - 우측 상단 **"Create form"** 클릭
   - 링크 복사 → 북마크 추가
   - 매일 이 링크로 데이터 입력!

---

## 💡 핵심 팁 3가지

### 1. Form이 핵심!
- 직접 테이블 수정하지 말고
- Form으로 입력하면 10배 빠름

### 2. View 활용
- Grid view 옆 **"+"** 클릭
- Filter/Sort로 원하는 데이터만 보기

### 3. 모바일 앱 필수
- 언제 어디서나 확인/입력
- 알림도 받을 수 있음
